那么这两种创建线程的方式哪种好呢？
答案是第二种方法好，即实现 Runnable 方法更好。因为在Java中只能单继承，一个类继承了 Thread 类之后就不能再继承其他的类了，而一个类却可以多实现，实现不同的接口，所以第二种方法比第一种更好。
