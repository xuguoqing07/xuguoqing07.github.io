现在需要统计每个学生的平时作业总得分。

首先准备一个文档，里面的数据如下：

诸葛亮 100

5.得到一个学生的总成绩后，把它添加到一个 list 中
6.最后，全部成绩处理完毕后，把results中的内容保存至文件
