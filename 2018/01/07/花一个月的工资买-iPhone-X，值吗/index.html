花一个月的工资买 iPhone X，值吗

发表于 2018-01-07 | 分类于 成长

读《富爸爸穷爸爸》

说来惭愧，最近才刚读完《富爸爸穷爸爸》这本理财书，整理这本书的资料的时候，突然想起来前不久很多人对新鲜出炉的 iPhone X 趋之若鹜，花费掉自己一个月甚至几个月的工资来购买。

iPhone X 肯定差不到哪儿去，但是它有没有好到我们可以用自己一个多月的工资去买一部这样的手机呢？

我觉得完全没有必要。

首先，很多人对于是否购买 iPhone X 非常纠结，这本身就说明自己的经济实力可能还没有达到一定的程度。这一点我就和他们不一样，我一点也不纠结，因为我买不起。

让我花一个月甚至几个月的工资去买一样东西也行，但是这样东西对我来说必须是刚需，或者说没有它我就见不到明天的太阳啦，没有它我以后就赚不了很多钱啦，没有它我很有可能找不到对象啦，那我还是会乖乖掏钱的。

但是，很显然，iPhone X 并不能提高生活质量，而且它很可能会拉低自己的生活水平。

过度消费自己的收入所带来的后果是可以预见的，这一个月或者以后的几个月你的生活会捉襟见肘。别人吃饭你喝水，别人吃肉你喝汤，别人谈笑风生你暗自神伤，只能抚摸着手里的 iPhone X 安慰自己这是精神食粮。

其次，人们之所以争相购买苹果手机新品，还有很大的原因在于攀比，以为这就是社会地位的象征。

有种手握苹果天下我有的感觉，认为这样别人就能对自己高看两眼，走在路上头都会比平时多抬高一度。

其实我们绝大多数人都是普通人，没人会在乎你用的什么手机，即使你手拿天价手机别人你只会对手机多看两眼，对你，人家没兴趣。

而且我也没见到卖水果的因为你用iPhone X 给你优惠很多钱，食堂大妈因为你的昂贵手机打饭的手再也不抖了，也没见过手拿iPhone X挤地铁别人连忙给让座的。

不论是苹果新版手机还是其他奢侈品都不是社会地位的体现，那些有钱人，社会地位高的人之所以使用一些昂贵的东西是由他们的财富和资产决定的，而并不是说你用了昂贵的东西就可以拥有像他们一样的财富。千万不要搞反了！

就像《富爸爸穷爸爸》书中所述，真正的奢侈品是对投资和积累资产的奖励。那些长期富裕的人，是先建立他们的资产，然后才用资产所产生的收入购买奢侈品，穷人和中产阶级则用他们的血汗钱购买奢侈品。

对大多数人成年人而言，要做的不是冲动的用贷款买奢侈品，而是把支出维持在低水平，减少借款和勤劳的工作会帮我们打下一个坚固的资产基础，为自己的事业或者未来增加筹码。

不满足是向上的车轮，谁都想拥有更好的东西。想用 iPhone X 或者其他奢侈品当然可以，那就去努力赚钱，培养自己的技能以及赚钱的本领。

我有时候就在想，现在世界变化越来越大，技术进步越来越快，大数据和人工智能整天充斥在我们耳边，慢慢变成了整个社会的主旋律，无数人包括我自己有时候都会感到焦躁不知前路如何。

但同时转念一想，这个时代提供给你和你的家庭乃至几代人的机会也越来越多，一些人仅凭思想和所谓的合约就能从一无所有到一夜暴富。

或许再过个若干年，后人回看我们现在的年代会觉得这是一个充满机遇的时期。所以，不如试着静下心来，做好自己的工作，多储蓄为以后蓄力，多培养赚钱的能力比什么都强。
